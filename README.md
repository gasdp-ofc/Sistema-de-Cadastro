# 📊 Sistema de Cadastro com Interface Gráfica

## 🚀 Descrição do Projeto
Sistema desenvolvido em Python com interface gráfica utilizando Tkinter, com o objetivo de simular um ambiente real de organização e gestão de dados.

O sistema permite cadastrar, consultar, buscar e remover registros, com armazenamento local em arquivo JSON.

---

## 🎯 Objetivo
Aplicar tecnologia para resolver um problema real de organização de dados, promovendo eficiência, praticidade e redução de erros manuais.

---

## 💻 Funcionalidades
- Cadastro de usuários
- Listagem de registros
- Busca por nome
- Remoção de cadastros
- Persistência de dados (JSON)
- Interface gráfica intuitiva

---

## 🛠️ Tecnologias utilizadas
- Python
- Tkinter
- JSON
- PyInstaller (para versão executável)

---

## 📦 Versão executável
O projeto pode ser executado como aplicativo independente através do arquivo `.exe` localizado na pasta `dist`.
